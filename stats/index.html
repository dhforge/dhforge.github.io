중요한 제한

GitHub Pages는 서버가 없는 정적 사이트라서 GA 실제 숫자를 이 HTML 파일이 직접 가져올 수 없습니다. GA API 비밀키를 공개 저장소에 넣으면 계정 보안 문제가 생깁니다. 그래서 실제 숫자는 아래 Looker Studio 보고서 안에 표와 카드로 만들어야 합니다.

지금 스크린샷처럼 조회수 카드 1개만 보이면 Looker Studio 보고서 편집 화면에서 카드와 표가 아직 추가되지 않은 상태입니다. 보고서에는 위 표의 항목대로 날짜, 사이트, 조회수, 방문자 수, 평균 방문시간, tool_use 이벤트 수, 사용률을 넣어야 합니다.
