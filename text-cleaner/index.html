텍스트 정리 도구

줄바꿈, 공백, 중복 줄, 대소문자를 빠르게 정리합니다.

텍스트 정리 도구를 쓰는 방법

복사한 문서, 표, 메모에서 불필요한 공백과 빈 줄을 정리합니다. 중복 줄 제거, 연속 공백 정리, 대소문자 변환을 한 화면에서 처리할 수 있습니다.

공백 정리

줄 앞뒤 공백이나 여러 번 반복된 공백을 읽기 쉬운 형태로 바꿉니다.

빈 줄 제거

붙여넣기 과정에서 생긴 불필요한 빈 줄을 빠르게 줄입니다.

중복 줄 제거

목록에서 같은 줄이 여러 번 들어간 경우 한 번만 남깁니다.

사용 순서

  1. 정리할 텍스트를 원본 입력칸에 붙여넣습니다.
  2. 필요한 정리 버튼을 누릅니다.
  3. 결과를 확인한 뒤 복사해서 사용합니다.

예시

  • 엑셀이나 문서에서 복사한 목록의 빈 줄을 제거할 때
  • 중복된 URL 목록을 하나씩만 남길 때
  • 영문 제목을 대문자나 소문자로 통일할 때
원본 텍스트가 자동으로 바뀌나요?

아니요. 결과 칸에만 정리된 텍스트가 표시됩니다.

중복 줄 제거는 순서를 유지하나요?

처음 나온 줄의 순서를 유지하면서 이후 같은 줄을 제거합니다.

텍스트가 저장되나요?

저장되지 않습니다. 모든 작업은 현재 브라우저 안에서만 실행됩니다.
