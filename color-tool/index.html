색상 변환기

HEX, RGB, HSL 값과 배경 대비를 확인합니다.

색상 변환기를 쓰는 방법

색상을 선택하거나 HEX 값을 입력하면 RGB, HSL 값과 검정/흰색 글자 대비를 확인할 수 있습니다. 웹 디자인, 문서 색상, UI 색상 조합을 검토할 때 유용합니다.

HEX 확인

웹에서 자주 쓰는 #2563eb 같은 6자리 HEX 색상 값을 확인합니다.

RGB/HSL 변환

CSS나 디자인 도구에서 필요한 RGB, HSL 값을 함께 보여줍니다.

대비 확인

선택한 배경색 위에서 검정 글자와 흰색 글자의 대비율을 비교합니다.

사용 순서

  1. 색상 선택기에서 색을 고르거나 HEX 값을 입력합니다.
  2. RGB, HSL, 대비율 결과를 확인합니다.
  3. 더 읽기 쉬운 글자색을 선택합니다.

예시

  • 버튼 배경색에 어울리는 글자색을 고를 때
  • 디자인 시안의 HEX 값을 CSS 값으로 바꿀 때
  • 색상 팔레트의 기본 정보를 빠르게 확인할 때
대비율은 무엇을 의미하나요?

배경색과 글자색의 밝기 차이를 비율로 나타낸 값입니다. 값이 높을수록 글자가 더 뚜렷하게 보입니다.

3자리 HEX도 지원하나요?

현재는 정확한 입력을 위해 6자리 HEX 형식을 기준으로 합니다.

색상 값이 저장되나요?

아니요. 선택한 색상은 현재 브라우저 화면에서만 계산됩니다.
