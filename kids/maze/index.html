Printable Activity

미로 찾기

시작부터 도착까지 항상 이어지는 무료 미로 찾기 프린트입니다.

미로 찾기 사용 방법

새로 만들 때마다 시작부터 도착까지 연결된 미로가 생성됩니다. 인쇄 창에서 프린터 또는 PDF 저장을 선택하세요.

무료 출력

브라우저 인쇄 기능으로 종이에 출력하거나 PDF로 저장할 수 있습니다.

가정과 교실

부모, 보호자, 교사가 짧은 연습 자료를 만들 때 사용할 수 있습니다.

로그인 없이 사용

이름이나 이메일을 입력하지 않아도 학습지를 만들고 출력할 수 있습니다.

어떤 용도로 쓰면 좋나요?

가정 학습, 수업 보조, 짧은 반복 연습, 숙제 전 준비 활동에 사용할 수 있습니다.

무료로 사용할 수 있나요?

네. 현재 페이지의 학습지 생성 기능은 무료로 사용할 수 있습니다.
