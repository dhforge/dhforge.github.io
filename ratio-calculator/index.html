화면 비율 계산기를 쓰는 방법

이미지, 영상, 썸네일 크기를 같은 비율로 줄이거나 키울 때 사용합니다. 원본 너비와 높이를 입력하면 간단한 비율과 새 크기를 계산합니다.

비율 단순화

1920x1080 같은 해상도를 16:9처럼 보기 쉬운 비율로 표시합니다.

너비 기준 계산

새 너비를 입력하면 같은 비율의 새 높이를 계산합니다.

높이 기준 계산

새 높이를 기준으로 필요한 너비도 확인할 수 있습니다.

사용 순서

  1. 원본 너비와 높이를 입력합니다.
  2. 새 너비 또는 새 높이를 입력합니다.
  3. 계산된 크기를 이미지 편집이나 영상 편집에 사용합니다.

예시

  • 유튜브 썸네일을 같은 비율로 줄일 때
  • 웹 배너 이미지를 특정 너비에 맞출 때
  • 사진을 찌그러지지 않게 리사이즈할 때
