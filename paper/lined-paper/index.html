Lined Paper Template

Create simple ruled paper for notes, letters, drafts, study pages, and writing practice. Pick the spacing and print a clean page.

How to use lined paper

Choose narrow, standard, or wide line spacing depending on the writing size. Add a title if the page is for a class, project, or journal entry.

Notes

Print quick note pages when you need clean ruled paper without buying a notebook.

Writing practice

Use wide spacing for larger handwriting and standard spacing for everyday notes.

PDF copy

Use your browser's print dialog to save the generated sheet as PDF.

Can I print multiple copies?

Yes. Set the copy count in the print dialog.

Does the page include margins?

The preview includes visual margins, and the final result also depends on printer margins.

Can I use this for school notes?

Yes. The template is suitable for ordinary study and classroom note pages.
