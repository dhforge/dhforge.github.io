모눈종이 생성기

수학 문제 풀이, 좌표 연습, 도면 스케치, 작은 계획표에 쓸 수 있는 정사각 격자 종이를 만듭니다. 종이 크기, 간격, 선 색상, 제목을 조정한 뒤 바로 출력하거나 PDF로 저장할 수 있습니다.

모눈종이 생성기 사용 방법

원하는 종이 크기와 간격을 선택하고, 필요하면 제목을 넣은 뒤 인쇄 버튼을 누르세요. 브라우저 인쇄 창에서 프린터를 선택하거나 PDF로 저장할 수 있습니다.

바로 출력

설정 변경이 미리보기에 즉시 반영되어 출력 전에 결과를 확인할 수 있습니다.

개인정보 부담 감소

제목과 설정은 현재 브라우저 탭에서 처리되며 사이트 자체 서버에 저장하지 않습니다.

반복 사용

자주 쓰는 양식은 PDF로 저장해 두고 필요할 때 다시 출력하면 편합니다.

활용 팁

  • 좌표, 그래프, 도형 넓이처럼 칸 기준이 필요한 수학 활동에 잘 맞습니다. 칸이 너무 촘촘하면 쓰기 어렵고, 너무 넓으면 한 장에 들어가는 정보가 줄어드니 보통 간격부터 테스트 출력하세요.
  • 스케치 용도라면 선 색상을 연하게 두면 실제 그림이나 메모가 더 잘 보입니다. PDF로 저장해 두면 같은 양식을 계속 다시 출력하기 쉽습니다.
  • 비슷한 용도라면 도트 그리드는 더 가볍고, 아이소메트릭 종이는 입체 스케치에 더 적합합니다.
무료로 사용할 수 있나요?

네. 개인, 학습, 일반적인 수업과 업무 정리 용도로 무료로 사용할 수 있습니다.

설정이 서버에 저장되나요?
