    How to use this timer and stopwatch

    Use the countdown for focused work, practice sessions, breaks, or workouts. Use the stopwatch when you need elapsed time and lap records.

    Countdown

    Enter minutes and track the time remaining.

    Stopwatch

    Measure elapsed time with a tenth-of-a-second display.

    Lap records

    Add lap entries to compare repeated tasks.

    Steps

    1. Enter minutes and click Start for a countdown.
    2. Click Start on the stopwatch to measure elapsed time.
    3. Use Lap to save intermediate records.

    Examples

    • Running a 25-minute focus session
    • Checking presentation practice time
    • Timing workout sets and breaks
    Does time continue after closing the browser?

    No. It only runs in the currently open browser tab.

    Does it play an alarm sound?

    No. This version is a quiet timer without an alarm sound.

    Are lap records saved?

    No. Refreshing the page clears the lap list.
