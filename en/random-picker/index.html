How to use this random picker

Enter one item per line to choose a random winner, shuffle the list, or split people into teams. You can also draw a random integer between a minimum and maximum value.

Name picker

Choose one item from the list at random.

Team splitter

Shuffle the list and distribute items across teams as evenly as possible.

Number picker

Pick a random whole number inside the selected range.

Steps

  1. Enter names or items, one per line.
  2. Choose pick, shuffle, or make teams.
  3. Use the number range if you need a random number.

Examples

  • Picking a presentation order
  • Splitting a group into teams
  • Drawing a giveaway number
Will the same list always give the same result?

No. Each click uses a fresh random selection from the browser.

Are entered names saved?

No. The list is only used in the current browser tab.

Can team sizes differ?

Yes. If the item count does not divide evenly, some teams can have one extra item.
