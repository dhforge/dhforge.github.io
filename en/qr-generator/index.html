QR Utility

QR Code Generator

Create a QR code in the browser without calling an external QR API.

Best for short URLs and brief text. Long text may be too large for this compact generator.

How to use this QR code generator

Enter a short URL or text, generate a QR code, and download it as a PNG file. The QR code is created with JavaScript and canvas inside the browser.

URL QR codes

Turn a website address into a QR code that can be opened by a phone camera.

Text QR codes

Encode short notes or simple values as a scannable image.

PNG download

Save the generated code for a document, sign, or handout.

Steps

  1. Enter the URL or text.
  2. Click Make QR to render the image.
  3. Use Save PNG if you need a file.

Examples

  • Adding a website link to a printed flyer
  • Sharing a short event page URL
  • Making a quick QR code for a simple text value
Can it handle long paragraphs?

This generator is tuned for short URLs and short text. Shorten long content first for best results.

Is the content sent to a QR API?

No. The QR code is generated in the current browser tab.

Can I use the generated QR commercially?

You may use the generated image, but you are responsible for the rights and accuracy of the link or text inside it.
