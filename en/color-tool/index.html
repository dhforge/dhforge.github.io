How to use this color converter

Choose a color or type a six-digit HEX value. The tool shows matching RGB and HSL values and compares the contrast for black and white text on that background.

Steps

  1. Select a color or enter HEX.
  2. Review RGB, HSL, and contrast values.
  3. Choose the text color with the clearer contrast.

Examples

  • Converting a design color to CSS values

What does contrast ratio mean?

It compares relative brightness between text and background. Higher values are easier to read.

Does it support three-digit HEX?

This version expects a six-digit HEX value for clarity.

Is the selected color saved?

No. Values are calculated in the current browser tab.
