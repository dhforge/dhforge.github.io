How to use this date calculator

Use the first section to count the days between two dates. Use the second section to add or subtract days from a base date. It is useful for deadlines, schedules, trip lengths, and personal milestones.

Date difference

Select a start and end date to see the number of days between them.

Inclusive count

The result also shows the count when the start date is included.

Add days

Enter positive or negative days to move forward or backward from a date.

Steps

  1. Select the start and end dates.
  2. Enter a base date and day offset if needed.
  3. Use the result in your calendar, schedule, or notes.

Examples

  • Finding the date 100 days from today
  • Counting days until a project deadline
  • Checking the length of a trip
Does it account for leap years?

Yes. It uses the browser's date handling, so leap years and month lengths are reflected.

Does it calculate business days?

No. Holidays and business days vary by country and year, so this tool only calculates calendar days.

Can I use negative day offsets?

Yes. A negative number calculates a date before the base date.
