Image Utility

이미지 리사이즈

이미지를 서버에 올리지 않고 브라우저에서 크기를 줄이고 저장합니다.

결과 저장

이미지 리사이즈 도구를 쓰는 방법

JPG, PNG, WebP 이미지를 브라우저에서 직접 불러와 원하는 크기로 변경합니다. 파일은 서버에 업로드하지 않고 현재 브라우저의 canvas에서 처리합니다.

크기 변경

너비와 높이를 입력해 이미지 크기를 조정합니다.

비율 유지

체크박스를 켜면 원본 비율에 맞춰 다른 한쪽 크기를 자동 계산합니다.

형식 선택

PNG, JPG, WebP 중 필요한 형식으로 저장할 수 있습니다.

사용 순서

  1. 이미지 파일을 선택합니다.
  2. 원하는 너비와 높이를 입력합니다.
  3. 형식을 선택하고 이미지 만들기를 누른 뒤 결과를 저장합니다.

예시

  • 블로그에 올릴 이미지를 작게 줄일 때
  • 프로필 이미지 크기를 맞출 때
  • PNG 이미지를 WebP로 저장해 용량을 줄이고 싶을 때
선택한 이미지가 업로드되나요?

아니요. 이 도구는 서버 업로드 없이 브라우저 안에서 이미지를 처리합니다.

큰 이미지도 사용할 수 있나요?

브라우저 메모리 한도 안에서는 사용할 수 있습니다. 매우 큰 이미지는 기기 성능에 따라 느릴 수 있습니다.

원본 파일이 바뀌나요?

아니요. 원본 파일은 그대로 두고 새 결과 이미지를 저장하는 방식입니다.
